## Running

### Installing dependencies

Using npm, install all dependencies:
```bash
$ npm i
```

## Running using npm

```bash
# For a local dev environment
$ npm run dev

# For a production build
$ npm run build

# Hosting your production build ( build a prod build first )
$ npm run preview
```
